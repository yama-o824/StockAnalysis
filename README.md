# StockAnalysis (v0.2)

WPFアプリからPython（yfinance）を起動し、株価データ（OHLCV）を取得し、
75日単純移動平均（MA75）を計算して表示するアプリケーションです。

------------------------------------------------------------------------

## v0.2 変更点

-   MA75（75日単純移動平均）を追加
-   75本未満は null（空欄）表示
-   価格系は小数2桁表示（N2）
-   出来高は整数表示（N0）
-   DataGridを ReadOnly に変更

------------------------------------------------------------------------

## 概要

-   WPF → Python プロセス起動
-   stdout の JSON を受け取り → DataGrid 表示
-   C#側で MA75 を計算
-   Python側で当日キャッシュ（同一銘柄・同一期間は再取得しない）

------------------------------------------------------------------------

## 動作環境

-   Windows 10 / 11
-   .NET 8 SDK
-   Python 3.10以上（推奨）

------------------------------------------------------------------------

## セットアップ手順

### 1. リポジトリをクローン

``` bash
git clone https://github.com/yama-o824/StockAnalysis.git
cd StockAnalysis
```

### 2. Python（fetcher）のセットアップ

``` bash
cd tools/fetcher
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

> python
> が見つからない場合は、PythonをインストールしてPATHを通してください。

### 3. WPFアプリを起動

Visual Studioで以下のプロジェクトを開いて実行してください。

    src/StockAnalyzer

------------------------------------------------------------------------

## 使い方

1.  銘柄コードを入力（例：7203.T）
2.  「取得」ボタンを押す
3.  OHLCVおよびMA75が表示されます

-   期間は v0.2 では固定（例：1y）
-   キャッシュは `tools/fetcher/cache`
    に保存されます（当日中は再取得しません）
