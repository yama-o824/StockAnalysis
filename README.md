# KabuAnalyzer (v0.1)

WPFアプリからPython（yfinance）を起動し、株価データ（OHLCV）を取得してDataGridに表示するサンプルです。

## 概要

-   WPF → Python プロセス起動
-   stdout の JSON を受け取り → DataGrid 表示
-   Python側で当日キャッシュ（同一銘柄・同一期間は再取得しない）

------------------------------------------------------------------------

## 動作環境

-   Windows 10 / 11
-   .NET 8 SDK
-   Python 3.10以上（推奨）

------------------------------------------------------------------------

## セットアップ手順

### 1. リポジトリをクローン

``` bash
git clone <this-repo>
cd KabuAnalyzer
```

### 2. Python（fetcher）のセットアップ

``` bash
cd tools/fetcher
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

> python
> が見つからない場合は、PythonをインストールしてPATHを通してください。

### 3. WPFアプリを起動

Visual Studioで `src/KabuAnalyzer.Wpf` を開いて実行してください。

------------------------------------------------------------------------

## 使い方

1.  銘柄コードを入力（例：7203.T）
2.  「取得」ボタンを押す
3.  DataGridに株価（OHLCV）が表示されます

-   期間は v0.1 では固定（例：1y）
-   キャッシュは `tools/fetcher/cache`
    に保存されます（当日中は再取得しません）

------------------------------------------------------------------------

## エラーが発生した場合

-   銘柄コードが誤っている可能性があります
-   アクセス集中（レート制限）の場合は時間を置いて再実行してください
-   Pythonライブラリが不足している場合は requirements.txt
    を再インストールしてください
